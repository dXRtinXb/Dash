# Nobel Prize Explorer

This project is a mini-application built with Python's Dash framework to visualize and explore the Nobel Prize dataset. It allows users to interactively filter and view information about Nobel laureates, their prizes, and the categories in which they were awarded.

## Features

*   **Interactive Visualization**: Explore Nobel Prize data through dynamic charts and tables.
*   **Filtering Capabilities**: Filter laureates by year, category, gender, and country.
*   **Data Overview**: Get a comprehensive look at Nobel Prize statistics.

## Technologies Used

*   **Python**: The primary programming language.
*   **Dash**: A Python framework for building analytical web applications.
    *   Dash Core Components (`dash_core_components`): For interactive elements like dropdowns, sliders, and graphs.
    *   Dash HTML Components (`dash_html_components`): For creating HTML layouts.
    *   Plotly (`plotly`): For generating interactive plots.
*   **Pandas**: For data manipulation and analysis.
